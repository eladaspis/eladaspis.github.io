# eladaspis.github.io

Personal academic website for Elad Aspis — PhD candidate at Ben Gurion University of the Negev.

## Setup

```bash
bundle install
bundle exec jekyll serve
```

## Customization

- `_config.yml` — site settings, bio, social links
- `_data/education.yml` — education history
- `_data/employment.yml` — work experience
- `_data/publications.yml` — research publications
- `_data/highlights.yml` — highlighted projects
- `_data/news.yml` — news items
- `images/portrait.jpg` — add your photo here
